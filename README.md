# 筋トレ管理

ジムでのトレーニングを効率化するPWA（Progressive Web App）ベースの筋トレ管理ツールです。

## 機能

- **今日のメニュー**: スケジュールに基づいて当日のメニューを自動表示。その場で追加・編集が可能。セット完了のチェック、メモ・身体情報の記録。
- **メニュー設定**: 曜日指定または日付指定でメニューを登録。メニュー名・重量・回数・セット数を管理。
- **カレンダー**: 過去のトレーニング履歴を確認。日付ごとの完了メニュー・総重量・身体情報を表示。月間サマリーでメニューごとの総重量を確認。

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
npm run preview
```

## PWAとしての利用

1. ビルドしたアプリをWebサーバーでホスト
2. スマートフォンのブラウザでアクセス
3. 「ホーム画面に追加」でアプリのように利用可能

## 技術スタック

- React 18 + TypeScript
- Vite
- React Router
- vite-plugin-pwa
- LocalStorage（オフライン対応）
